# 修改器异种金蝎三线并行方案

## 源

https://www.bilibili.com/opus/995870562260615176

## 工具

使用[CE表](https://bbs.3dmgame.com/forum.php?mod=viewthread&tid=6454278)，或者[一修大师](https://www.lmaoyx.com/pc/yixiu)。

## 物品ID

- 异种金蝎 ID：10498

- 人面蛛 ID：10488

## 流程

::: warning 注意

**先不要去【猴儿林】【无名岩洞】触发【娜乌】任务，否则其他两线不会触发！**

:::

1. 【清河村】【铁蛋】触发【支线】毒蝎夺命，完成该支线接到【支线】异种金蝎。

2. 修改出第一只【异种金蝎】（1），至【幽云泽】【元疆】触发【支线】毒物商人，**选择回绝**。

3【姑苏城】【柳麻衣】交付【异种金蝎】（1），此时接到【支线】药圣之约，**但是不要去做**。

> 同时还会显示【支线】毒物商人失败，但是不影响。

4. 立马至【幽云泽】修改出第二只【异种金蝎】（2），交付【元疆】。离开路上接到【支线】人面鬼蛛，修改出【人面蛛】直接交付，接到后续任务【支线】古卷帛书，商人线可正常继续了。

5.【猴儿林】【无名岩洞】击杀【异种金蝎】（3），至洞口遇见【娜乌】，选择赠与，触发【支线】飞蝎使，娜乌正常继续。

6.【姑苏城】【柳麻衣】完成【支线】药圣之约，药圣线正常继续。

::: tip 建议

建议一次性操作完上述流程，避免分开操作时有所遗漏。

:::
